Detailed Description

This class is responsible for the Miscellaneous Configuration management.

In this class are stored all the variables for MISC configuration.
You can set audio buffer, mouse speed, the tile and subtitle of the window.
You can also set the automatic grabbing of mouse and keyboard and the settings
of network game. Finally you can config the path of alternative directories.
