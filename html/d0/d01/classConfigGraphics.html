Detailed Description

This class is responsible for the Graphics Configuration management.

In this class are stored all the variables for GRAPHICS configuration.
You can select the DISPLAY SETTINGS, FILTERS, SHADERS AND SCALING, VIDEO SYNCHRONIZATION, and VIDEO AND TEXTURE BUFFER FORMAT.
DISPLAY SETTINGS: you can change for startup fullscreen mode and make possible the resizing window, change fullscreen and window resolution and enable full-scene anti-aliasing.
FILTERS, SHADERS AND SCALING: you can edit scanlines parameter, type of shaders, resolution and line doubling.
Here are also managed the position and alignment of the video, textures setting as well as video initial mode .
VIDEO SYNCHRONIZATION: here you can select type and methode of syncronization.
VIDEO AND TEXTURE BUFFER FORMAT: here you can select video and texture format.
