Detailed Description

This class is responsible for the Floppy Configuration management.

In this class are stored all the variables for FLOPPY configuration.
You can load up to a maximum of 4 physical floppy disk, and create a list with up to 20 floppy images to enable floppy-swapping during emulation without having to manually change the configuration file.
You can also set he output volume of the floppy disk drive and floppy drive speed.

This class is responsible for the Floppy Configuration.
