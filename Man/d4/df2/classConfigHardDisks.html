Detailed Description

This class is responsible for the Hard Disks Configuration management.

In this class are stored all the variables for HARD DISK configuration.
You can set 10 hard disk drive selecting a hdf file or using a directory
as a drvie. For each hard disk you can set a Label and set it as Read only.
